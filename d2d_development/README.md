# d2d_development

DHIS2-to-DHIS2 development utility library maintained by Bluesquare Data Services team.

## Installation

Install this library on its own:

```bash
pip install git+https://github.com/BLSQ/openhexa-ds-developments.git#subdirectory=d2d_development
```

Or install the full `openhexa-ds-developments` repo to get all libraries at once:

```bash
pip install git+https://github.com/BLSQ/openhexa-ds-developments.git
```

## Usage

```python
from d2d_development import ...
```
